# Дневник экологических привычек

Мобильное приложение для формирования и отслеживания экологичных привычек. Построено на Flutter с использованием Firebase.

## Возможности

### Основные функции
- **Регистрация и вход** — создание аккаунта по email, восстановление пароля
- **Каталог из 52 экологических привычек** — 6 категорий: экономия воды, энергии, обращение с отходами, транспорт, потребление, забота о природе
- **Ежедневный чек-лист** — отмечайте привычки одним касанием
- **Напоминания** — push-уведомления в заданное время
- **Статистика** — графики выполнения за неделю и месяц, серии, процент выполнения
- **Расчёт экологического эффекта** — вода, энергия, CO₂ с конвертацией в наглядные эквиваленты
- **Система достижений** — 24 достижения с анимациями разблокировки
- **Профиль и настройки** — тема, уведомления, язык, пароль

### Социальные функции
- **Друзья** — поиск по email/нику, отправка и принятие заявок в друзья
- **Лента друзей** — просмотр прогресса друзей: серии, процент выполнения, новые достижения
- **Таблица лидеров** — рейтинг по недельному/месячному прогрессу среди друзей
- **Шеринг достижений** — кнопка «Поделиться» на карточке достижения (в соцсети, мессенджеры)
- **Командные челленджи** — создание челленджей (до 5 участников), общий прогресс

## Методика расчёта экологического эффекта

Каждая привычка имеет коэффициенты, основанные на усреднённых научных данных:

- **Вода** — литры сохранённой воды (на основе данных UNESCO, EPA)
- **Энергия** — кВт·ч сэкономленной электроэнергии (на основе данных IEA)
- **CO₂** — килограммы предотвращённых выбросов (на основе данных IPCC)

### Визуальные эквиваленты:
- 🌳 Деревья = CO₂ / 22 кг (поглощение одним деревом в год)
- 🚗 Км на авто = CO₂ / 0.12 кг (выбросы среднего автомобиля)
- 💡 Часы LED = Энергия / 0.01 кВт·ч (потребление LED-лампы 10W)
- 🛁 Ванны = Вода / 150 л (объём стандартной ванны)

## Руководство пользователя

### Начало работы
1. Запустите приложение и создайте аккаунт
2. Перейдите во вкладку «Каталог»
3. Выберите привычки из 6 категорий
4. Настройте периодичность и напоминания

### Ежедневное использование
1. На главном экране отмечайте выполненные привычки
2. Переключайтесь между днями стрелками
3. Следите за прогрессом в статистике

### Достижения
- Выполняйте привычки подряд для разблокировки достижений
- 24 достижения от «Первого шага» до «Года зелёных привычек»
